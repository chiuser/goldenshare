# 大宗交易

- 分类：股票数据 / 参考数据
- 原始链接：[https://tushare.pro/document/2?doc_id=161](https://tushare.pro/document/2?doc_id=161)

- 接口：block_trade
- 描述：大宗交易
- 限量：单次最大1000条，总量不限制
- 积分：300积分可调取，每分钟内限制次数，超过5000积分频次相对较高，具体请参阅[积分获取办法](https://tushare.pro/document/1?doc_id=13)

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| ts_code | str | N | TS代码（股票代码和日期至少输入一个参数） |
| trade_date | str | N | 交易日期（格式：YYYYMMDD，下同） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| limit | int | N | 单次返回数据长度 |
| offset | int | N | 请求数据的开始位移量 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
| --- | --- | --- | --- |
| ts_code | str | Y | TS代码 |
| trade_date | str | Y | 交易日历 |
| price | float | Y | 成交价 |
| vol | float | Y | 成交量（万股） |
| amount | float | Y | 成交金额 |
| buyer | str | Y | 买方营业部 |
| seller | str | Y | 卖方营业部 |

## 接口用法

```python
pro = ts.pro_api()

df = pro.block_trade(trade_date='20181227')
```
